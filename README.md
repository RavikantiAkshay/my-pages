# Dev Archive | Akshay Ravikanti

A curated "showcase" gallery of personal mini/basic projects, basic web pages, and developmental experiments. This repository serves as a record of progress across various disciplines of frontend web development, from pure static structures to complex, logic-driven dynamic interfaces.

## 🚀 Project Overview

This is an organized index of foundational web development exercises and mini-pages. The collection is categorized into three distinct tiers of complexity:

### 📁 Category 01: Static Webpages
*Pure structure and semantic clarity. High-performance layouts built with HTML5 and Vanilla CSS.*
- **lifestyle & Wellness**: Yoga, Sleep Music, Happy Meals
- **Entertainment & Media**: Finding Nemo, Podcasts, Popular Book, VR Cricket
- **Tech & Innovation**: Advanced Tech, iPhone Specs, Conference
- **Travel & Hospitality**: Goa Tourism, Hotel, Sunrise Avenue
- **Education & Knowledge**: Foundations Courses, Quiz, Articles
- **Social & Commerce**: Rahul Chat, Favorite Stores, My Showcase
- **Profiles & Awareness**: KL Rahul, Covid Awareness, Todolist

### 📁 Category 02: Responsive Webpages
*Fluid systems built to adapt across devices. Utilizing Bootstrap and advanced CSS layouts.*
- **Assembly Projects**: Food Munch, VR Revolution, Portfolio Website, Fly Buy Ecommerce
- **Components**: Move Messenger, Inspiring Quotes, Travel Benefits, Design Section, Pricing Plans, Robotics, Dream Car

### 📁 Category 03: Dynamic Webpages
*Logic-driven interfaces that respond to real-time data using the Fetch API and State Management.*
- **Varanasi Experience**: Tabbed state management.
- **Joke Engine**: Real-time humor generator (Fetch API).
- **Wiki Search Engine**: Asynchronous search tool (Wiki API).
- **Population Pulse**: Global country data search.
- **Number Secrets**: Mathematical trivia explorer.
- **API Console**: REST testing tool for POST/PUT requests.

## 🛠️ Tech Stack
- **Languages**: HTML5, CSS3, JavaScript (ES6+)
- **Styling**: Vanilla CSS, TailwindCSS (Landing Page), Bootstrap (Responsive Category)
- **APIs**: Wikipedia API, REST APIs, Fetch API for dynamic data
- **Icons & Fonts**: Google Fonts, Material Symbols, FontAwesome

## 📖 How to Use
1.  **Clone the Repository**: `git clone https://github.com/RavikantiAkshay/my-pages`
2.  **Open the Landing Page**: Simply open `index.html` in any modern browser to navigate through the curated gallery.
3.  **Explore Isolated Projects**: Each project is contained within its own directory under `projects/` and can be run independently.
